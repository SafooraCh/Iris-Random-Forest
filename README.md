# Task  — Iris Random Forest (Streamlit)

This project trains a Random Forest classifier on the Iris dataset (in Kaggle) and deploys the trained model using Streamlit.

## Files
- `app.py` — Streamlit app
- `rf_iris_model.pkl` — trained model bundle exported from Kaggle
- `requirements.txt` — dependencies

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
